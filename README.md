# pi-extensions

A collection of [pi](https://github.com/earendil-works/pi-mono) agent extensions I made.

- [`pi-worktree`](./packages/pi-worktree): Adds `/worktree`, which creates, switches, lists, and cleans up Git worktrees from inside pi.

## Install

Full collection:

```bash
pi install git:github.com/jastro-dev/pi-extensions
```

Then reload pi:

```text
/reload
```
