# Changelog

## pi-worktree-v0.1.0

- Added the initial `pi-worktree` extension with `/worktree` commands for creating, switching, listing, scratching, and cleaning Git worktrees from pi.
- Added GitHub issue and pull request URL expansion for `/worktree create` and `/worktree ccd`, including generated worktree titles and branch names.
- Added git-based pi package installation support for the monorepo.
